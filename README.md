# Weather Data Visualization Project

## Project Overview

This project explores the visualization of weather data using Python. The dataset used is from the Python Crash course by Eric Matthes.


## Data Source
Weather Data from Python Crash book
link below:
[Data source](https://github.com/ehmatthes/pcc_3e/tree/main/chapter_16/the_csv_file_format/weather_data)


## Visualization

1. Temperature over time
2. Precipitation Distribution

## Technologies Used
1. Python
2. Matplotlib
3. Seaborn
4. Pandas
